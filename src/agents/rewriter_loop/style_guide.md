# 风格指南（style_guide）— v1

> 本文件是写作子智能体与 linter 的**单一事实源（SSoT）**：
> 注入 writer 的 system prompt（散文部分）与 linter 解析加载的词表/禁用词/boilerplate（下方 YAML 块）同处一文件，零漂移。
> 已覆盖公文范式 base + 本科/高职双子风格 + 子风格裁定（学制/培养规格/职业面向/实践环节）
> + 价值与意识形态内容（(A) 政治理论表述逐字/归位/双向 + (B) 情感语在位/略松，含金融领域专属标签）
> + 引用事实校验（在位 + 查臆造：行业代码/学分/证书名按类型正则抽 token 比对 references）
> + 素材引用角标校验（本项目单方括号 `[素材id]` 语义：无杜撰角标/池内 id 唯一/回链假说/照抄须挂角标）。

## 公文范式 base（跨子风格）

- 章节用中文数字编号：`## 一、` `## 二、` …（禁用阿拉伯数字+顿号起手，如 `1、`）。
- 章下设 `### （一）` `### （二）` 子节；子节下设 `#### 1.` `#### 2.` 或 `1)` `2)` 条目。
- 段落行政语感，禁讲客体（不出现「我们」「咱们」「上一章讲到」「接下来」「下面我们」「本章我们」）。
- 该有表格的章须含 markdown 表（`| ... |` + 分隔行 `| --- | --- |`）。
- 公文用第三人称/无主语祈使（「本专业培养…」「根据…确定」），衔接上一章用公文风格承上启下，禁口语化过渡。
- **密表章 do-don't**（职业面向/培养目标与培养规格/课程设置等表型章）：
  - **do**：outline_entry.points 为子节标题时逐个产出 `### （N）` 子节；表须完整——列头按语料该章型、每条目一行、多值单元格用 `<br>` 换行；含注脚的章型保留 `注：…依据《…》` 引用标准；证书列头用「职业技能等级证书举例」并取 references 精确值。
  - **don't**：不合并子节、不以叙述段替表、不简化行列、不加叙述性凑段、不用「可考取证书」等口语化替法。

## 本科变体（有据）

- **学制学位**章：措辞「标准学制X年，授予X学士学位」；章标题作「学制学位」。
  本科禁混入高职措辞「基本修业年限」、不授予学位的高职表述。
- **培养规格**子项：本科**四分**（思政/素质/知识/能力，思政独立成项）。
  - **培养目标与培养规格**章结构：`### （一）培养目标` = **一段行政公文长段**（践行社会主义核心价值观、德智体美劳全面发展、工匠精神/报国情怀、面向XX行业的XX型人才），**不**写成工程认证式「毕业后五年目标」分点列表；`### （二）培养规格` = `#### 1.思政要求` `#### 2.素质要求` `#### 3.知识要求` `#### 4.能力要求` 四个子节，各以密集长段或 `1)xxx<br>2)xxx` 编号条目展开（**不**压缩成单一简表），能力要求子节可含 `|职业能力|职业能力解构|` 式表。
- **职业面向**章：大章权威词用「职业面向」；「职业领域」仅作局部小节名，不作大章标题。
- **实践环节**：用「实践教学」「实践环节」表述；集中性实践称「集中实践课程」「综合实践」（本科口径见语料实证，边界裁定见下方「实践环节裁定」）。

## 高职变体（有据）

- **学制**章：措辞「基本修业年限：X年」；章标题作「基本修业年限」（linter 别名归并到学制学位模板）。
  高职不授学位，禁混入本科措辞「标准学制」「学士学位」。
- **培养规格**子项：高职**三分**（素质/知识/能力，思政并入素质，不设独立思政子项）。
- **职业面向**章：大章权威词与本科一致用「职业面向」；「职业领域」仅作局部小节名。
- **实践环节**：高职语料多用「实践教学」「实践性教学环节」「综合实训」；裁定见下方。

## 实践环节裁定（子风格边界，语料实证）

- **实践教学**：作为总称性上位词，指代一切非理论教学的实践类环节（本科/高职通用）。
- **实践环节**：与「实践教学」近义，多见于培养规格与实施保障章的条目化表述。
- **集中实践课程**（本科口径）：以周为单位的集中性实践单元，归入课程体系。
- **综合实践 / 综合实训**（高职口径）：毕业阶段的综合性质实践，常含顶岗实习与毕业设计。
  本科称「综合实践」、高职称「综合实训」为语料常见分布；作章标题时优先取「职业面向」式权威词而避免裸用「实践环节」作大章标题。

## 术语 GLOSSARY（含 _Avoid_ 别名）

- **职业面向** — 大章权威词；_Avoid_ 作大章标题：`职业领域`（本科/高职通用）。
- **标准学制** — 本科措辞；_Avoid_ 别名：`学制年限`（本科）；高职禁用本科措辞见下方双向裁定。
- **基本修业年限** — 高职措辞；_Avoid_ 别名（高职禁用本科措辞）：`标准学制`；本科禁用此词见下方双向裁定。
- 子风格措辞纯度双向裁定（linter ``forbidden_terms`` 强制）：本科学制学位章禁出现「基本修业年限」；高职学制章禁出现「标准学制」「学士学位」。

## 连接性 boilerplate 句库

- 学制学位章（本科）：「标准学制」「授予…学士学位」。
- 学制章（高职）：「基本修业年限」。
- 职业面向章：「对应行业」「依据《国民经济行业分类》」。
  - **培养岗位类别表**（`### （一）培养岗位类别`）列头：`对应行业(代码)|主要职业类别(代码)|主要岗位类别(或技术领域)举例|职业技能等级证书举例`，行业/职业类别带括号代码、证书单元格用 `1.xxx<br>2.xxx` 编号密集列表。
  - **培养岗位内容表**（`### （二）培养岗位内容`）列头：`主要岗位类别(或技术领域)举例|工作任务|职业能力`，工作任务与职业能力单元格用编号密集列表 `1.xxx<br>2.xxx`，**不**以「证书」替「职业能力」列。
  - 两表后视语料保留 `注：(1)对应行业(代码)：依据《国民经济行业分类与代码》` 等注脚。
- 培养规格章：本科「思政要求」；高职「素质」起手。

## 素材引用与角标（本项目语义）

正文引用素材一律使用单方括号角标 `[素材id]`（如 `[m-h-1]`），id 只含字母数字、下划线、连字符。
正文角标的 id 必须存在于本章素材池内，禁止杜撰或篡改素材来源。
素材池内 id 必须唯一，素材身份由上游分配、不得重复登记。
素材必须回链本章既有假说，`hypothesis_id` 指向不存在的假说即为悬空。
照抄素材 `excerpt` 原文的正文位置必须挂对应 `[素材id]` 角标，不得未标派生。
参考文献由书目渲染层统一处理，正文不维护章内参考文献列表。

## 价值与意识形态内容（单列，语料实证）

意识形态内容是公文最高风险类，分两层：

### (A) 政治理论表述 — 逐字、归位、精确匹配

- **逐字复现**：`习近平新时代中国特色社会主义思想` `社会主义核心价值观` `四个自信` `立德树人` 等须逐字精确出现，不得改写或引错（如把「习近平新时代中国特色社会主义思想」简写为「习近平中国特色社会主义」属违规）。
- **归位**（哪章哪节）：政治理论表述只在该出现的归位章出现——
  - `社会主义核心价值观` 归位「培养目标与培养规格」章（培养目标指导思想句 + 培养规格思政要求），必含；课程设置章课程内容（如「思想道德与法治」课）亦见，允许。
  - `四个自信` 允许出现于本科「培养目标与培养规格」章但不强制（语料实证：非每篇本科培养目标章必含）。
  - `习近平新时代中国特色社会主义思想` 归位「课程设置」章（通识必修「概论」课），必含；培养目标章指导思想句、教学进程章排课表亦见，允许不强制。
  - `立德树人` 在「培养目标与培养规格」章允许出现不强制（各语料分布不一）。
- **章标题变体归并**：归位用规范章名（``chapter_groups``）匹配，使「培养目标及规格」「培养目标与四大培养规格」←「培养目标与培养规格」、「课程设置及要求」「课程设置及学时安排」「课程体系与学分学时总览」「课程架构」←「课程设置」、「教学进程总体安排」「教学周期安排」←「教学进程」同样受双向校验。
- **双向校验**：归位章内必含、归位章外禁出（不得幻觉注入到职业面向/学制等不相关章）。
- **领域标签**：`金融强国` `以义取利` 为金融领域专属政治语，归位金融方案「培养目标与培养规格」章且必含；非金融方案禁出。

### (B) 价值导向/情感语 — 在位校验、略松

- `工匠精神` `报国` `爱国` `强国志` 等价值导向语须在「培养目标与培养规格」章在位。
- 略松：容忍小幅措辞变化（正则任一命中即过，如「科技报国意识」「爱国情怀」「精益求精的工匠精神」均算在位）。
