# 风格指南 · 人才培养方案（文种层）

> 本文件是人才培养方案的文种层，与「通用公文」通用层两层合并加载（列表类并集、标量类文种覆盖，ADR-0005）。
> 本科/高职是本文种内两个变体（承接原 tier 概念），学制学位/培养规格等互斥措辞由 lint 按变体双向兜底。
> 禁学术论文断言句式（「XX是XX的必要条件」「XX正向预测XX」，lint 兜底），公文用行政表述（「以XX为基础」「XX有助于XX」）。

## 密表章 do-don't（职业面向/培养目标与培养规格/课程设置等表型章）

- **do**：outline_entry.points 为子节标题时逐个产出 `### （N）` 子节；表须完整——列头按语料该章型、每条目一行、多值单元格用 `<br>` 换行；含注脚的章型保留 `注：…依据《…》` 引用标准；证书列头用「职业技能等级证书举例」并取 references 精确值。
- **don't**：不合并子节、不以叙述段替表、不简化行列、不加叙述性凑段、不用「可考取证书」等口语化替法。
- **培养岗位类别表**（`### （一）培养岗位类别`）列头：`对应行业(代码)|主要职业类别(代码)|主要岗位类别(或技术领域)举例|职业技能等级证书举例`，行业/职业类别带括号代码、证书单元格用 `1.xxx<br>2.xxx` 编号密集列表。
- **培养岗位内容表**（`### （二）培养岗位内容`）列头：`主要岗位类别(或技术领域)举例|工作任务|职业能力`，工作任务与职业能力单元格用编号密集列表 `1.xxx<br>2.xxx`，**不**以「证书」替「职业能力」列。
- 两表后视语料保留 `注：(1)对应行业(代码)：依据《国民经济行业分类与代码》` 等注脚。

## 本科变体（有据）

- **学制学位**章：措辞「标准学制X年，授予X学士学位」，章标题作「学制学位」（禁用词/必含词由 lint 按变体兜底）。
- **培养规格**子项：本科**四分**（思政/素质/知识/能力，思政独立成项）。
  - **培养目标与培养规格**章结构：`### （一）培养目标` = **一段行政公文长段**（践行社会主义核心价值观、德智体美劳全面发展、工匠精神/报国情怀、面向XX行业的XX型人才），**不**写成工程认证式「毕业后五年目标」分点列表；`### （二）培养规格` = `#### 1.思政要求` `#### 2.素质要求` `#### 3.知识要求` `#### 4.能力要求` 四个子节，各以密集长段或 `1)xxx<br>2)xxx` 编号条目展开（**不**压缩成单一简表），能力要求子节可含 `|职业能力|职业能力解构|` 式表。
- **职业面向**章：大章权威词用「职业面向」；「职业领域」仅作局部小节名，不作大章标题。

## 高职变体（有据）

- **学制**章：措辞「基本修业年限：X年」，章标题作「基本修业年限」；高职不授学位，不得混入本科学制/学位措辞（lint 双向兜底）。
- **培养规格**子项：高职**三分**（素质/知识/能力，思政并入素质，不设独立思政子项）。
- **职业面向**章：大章权威词与本科一致用「职业面向」。

## 实践环节裁定（变体边界，语料实证）

- **实践教学**：总称性上位词，指代一切非理论教学的实践类环节（本科/高职通用）；**实践环节**与之近义，多见于条目化表述。
- **集中实践课程**（本科口径）：以周为单位的集中性实践单元，归入课程体系。
- **综合实践 / 综合实训**（高职口径）：毕业阶段的综合性质实践，常含顶岗实习与毕业设计。
  本科称「综合实践」、高职称「综合实训」为语料常见分布；作章标题时优先取「职业面向」式权威词而避免裸用「实践环节」作大章标题。

## few-shot 范例章（逐字整段，与 golden 测试章不相交）

> 取自《现代通信工程专业2025级本科人才培养方案》`## 二、专业基础信息`，与本仓库 golden 测试章（本科软件工程技术 `## 五、职业面向`、高职现代物流管理 `## 七、培养规格`）分属不同文档、不同章型，章节级不相交。

```text
## 二、专业基础信息
1. **专业名称**：现代通信工程
2. **专业代码**：310301
3. **入学要求**：普通高中/中职毕业或同等学力
4. **标准学制**：4年
5. **授予学位**：工学学士学位
6. **适用年级**：2025级
```

## 价值与意识形态内容（单列，语料实证）

意识形态内容是公文最高风险类，分两层（词表与归位章见 YAML，逐字/归位/双向/领域标签由 lint 兜底）：

- **(A) 政治理论表述**：`习近平新时代中国特色社会主义思想` `社会主义核心价值观` 等须**逐字精确复现**，不得改写或引错；且只在归位章出现（如「社会主义核心价值观」归位培养目标章必含），不得幻觉注入职业面向/学制等不相关章；`金融强国` `以义取利` 为金融领域专属，非金融方案禁出。
- **(B) 价值导向/情感语**：`工匠精神` `报国` `爱国` 等须在「培养目标与培养规格」章在位，容忍小幅措辞变化（「科技报国意识」「爱国情怀」均算在位）。
