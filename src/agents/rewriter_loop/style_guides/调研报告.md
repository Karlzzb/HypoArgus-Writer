# 风格指南 · 调研报告（文种层）

> 本文件是调研报告的文种层，与「通用公文」通用层两层合并加载（列表类并集、标量类文种覆盖，ADR-0005）。
> 章型（监测概述与数据说明 / 维度章 / 主要发现与问题诊断 / 结论与对策建议）由 State 随章携带，措辞与表格要求由 lint 按章型兜底。

## 实证分析语体（规范源「书写模式」）

- 图表驱动、文字精简：能用表格呈现的数据绝不用大段文字，文字仅用于解读数据背后的含义；维度章须含数据表（lint 兜底）。
- 中性客观、不带情绪：不写「令人振奋」「深感担忧」等主观形容词（lint 兜底），只写「数据显示…，比去年下降X个百分点」。
- 对比叙事：全文贯穿横向比（与兄弟院校/全省平均）、纵向比（与本校过去3-5年）、标准比（与预警线/合格线）三个对比维度。
- 强逻辑链条：每章遵循「数据呈现 → 现象归纳 → 归因分析 → 对策建议」递进，对策只在末章给出，前部章节不前置对策建议（lint 兜底）。
- 禁学术论文断言句式（「XX是XX的必要条件」「XX正向预测XX」，lint 兜底），用「数据显示」「数据表明」式实证陈述。

## 章型要点

- 维度章标题为凝练的观点标题（如「空间下沉：精准破局『基层人才荒』」），章内以数据观察结论 + 数据支撑展开。
- 主要发现与问题诊断：领先优势提炼核心亮点，不足与短板每条附数据证据，归因分析结合数据交叉分析提出病因猜想。
- 结论与对策建议：核心结论一句话凝练全篇；每条对策建议须明责任（牵头/落实部门）、定时限（完成年月或期限，lint 兜底），范式「建议一：…（教务处牵头，各二级学院落实，2026年9月前启动）」。
