# 风格指南 · 通用公文（跨文种通用规则层）

> 本文件双重身份（ADR-0005）：既是「通用公文」兑底文种的风格指南，也是跨文种通用规则层。
> 任何文种都先加载本文件，再叠加文种自身指南——列表类规则取并集，标量类规则文种覆盖通用；
> 兑底文种（自由结构 / 未登记模板）与暂无专属指南的文种只加载本文件。
> 散文注入 writer 的 system prompt，下方 YAML 块由 style_linter 解析加载，同一文件双消费、零漂移。

## 公文范式（跨文种）

- 章节用中文数字编号：`## 一、` `## 二、`…；章下设 `### （一）` `### （二）` 子节，子节下设 `#### 1.` 或 `1)` 条目（阿拉伯数字+顿号起手由 lint 兜底禁用）。
- 公文用第三人称/无主语祈使（「本专业培养…」「根据…确定」），禁讲客体与口语化过渡（词表由 lint 兜底）。
- 衔接上一章用公文风格承上启下，禁口语化过渡。
- 该有表格的章须含 markdown 表（`| ... |` + 分隔行 `| --- | --- |`）。

## 素材引用与角标（本项目语义）

- 正文引用素材一律用单方括号角标 `[素材id]`（如 `[m-h-1]`）；只可引用素材池内 id，禁止杜撰或篡改来源。
- 素材须回链本章既有假说；照抄素材原文的正文位置必须挂对应角标。
- 量化断言（提升/降低/缩短/增长/减少 + 数值单位）须同句挂素材角标、或其数值有参考依据支撑，否则视为臆造；表内数字由表承载，不在此列。
- 参考文献由书目渲染层统一处理，正文不维护章内参考文献列表。

## 字数管控（三级区间，散文统计口径）

- 统计口径：散文字数不含 markdown 表格、素材角标、公式、围栏代码块、参考文献与附录；标题行计入。
- 章/节/小节三级区间与动态收缩由 lint 兜底，是机检硬标准；每章目标区间随任务注入，成文前须逐级自查各章节字数落在区间内。
- 表章豁免散文下限：信息由表承载，散文只做引言与解读，不得在表外堆砌叙述性段落凑字数。
